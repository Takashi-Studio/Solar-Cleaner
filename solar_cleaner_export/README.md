# ☀️ دليل تشغيل نظام تنظيف الألواح الشمسية الذكي (SaaS Setup Guide)

نظام متكامل لتنظيف ومسح الألواح الشمسية تلقائياً وعن بُعد، مبرمج باستخدام **Arduino Mega 2560** كعقل مدبر و **ESP-01** كبوابة إنترنت ذكية، مدعوماً بسيرفر SaaS مكتوب بـ **Node.js (TypeScript) + SQLite** ولوحة تحكم ويب تفاعلية بـ **React + Tailwind CSS**.

---

## 🔌 أولاً: توصيل العتاد والمكونات (Hardware Connections)

### 1. توصيل قطعة الشبكة ESP-01 بالأردوينو ميقا:
*بما أننا قمنا ببرمجة ESP-01 مباشرة، سنقوم بربطها عبر منفذ الـ Hardware Serial1 المتواجد في الأردوينو ميقا:*
*   **VCC** (في ESP-01) 👈 منفذ **3.3V** في الأردوينو (تنبيه: لا توصله بـ 5V!).
*   **GND** (في ESP-01) 👈 منفذ **GND** في الأردوينو.
*   **RX** (في ESP-01) 👈 منفذ **TX1 (Pin 18)** في الأردوينو ميقا.
*   **TX** (في ESP-01) 👈 منفذ **RX1 (Pin 19)** في الأردوينو ميقا.
*   **CH_PD** (أو EN في ESP-01) 👈 منفذ **3.3V** (لتشغيل وتفعيل القطعة).

### 2. توصيل الحساسات والمحركات بالملحقات:
*   **حساس المسافة الخزان (HC-SR04):**
    *   `Trig Pin` 👈 منفذ **7** في الأردوينو.
    *   `Echo Pin` 👈 منفذ **8** في الأردوينو.
    *   `VCC` 👈 منفذ `5V` و `GND` 👈 `GND`.
*   **ريلي مضخة المياه 12V:**
    *   `Control Pin` 👈 منفذ **9** في الأردوينو.
*   **مفتاح نهاية الشوط (Limit Switch):**
    *   أحد الأطراف 👈 منفذ **10** في الأردوينو.
    *   الطرف الآخر 👈 منفذ **GND**.
*   **محركات الستيبر (Drivers - A4988 / TB6600):**
    *   المحرك الأول: `Step` 👈 منفذ **2**، و `Dir` 👈 منفذ **3**.
    *   المحرك الثاني: `Step` 👈 منفذ **4**، و `Dir` 👈 منفذ **5**.

## 🌐 ثالثاً: تشغيل وضبط الـ SaaS (سيرفر الـ Backend والـ Frontend)

### 1. إعداد خادم الـ Backend (Node.js + SQLite):
1. افتح نافذة الأوامر (Terminal) في مجلد `backend`:
   ```bash
   cd backend
   ```
2. تثبيت المكتبات البرمجية المطلوبة:
   ```bash
   npm install
   ```
3. تشغيل السيرفر في وضع المطور (سيقوم بإنشاء قاعدة بيانات `database.sqlite` تلقائياً فوراً):
   ```bash
   npm run dev
   ```
*سيعمل السيرفر على البورت `5000`.*

### 2. تثبيت وإعداد وسيط الـ MQTT (على السيرفر أو الـ VPS):
* يجب أن يتوفر وسيط **Mosquitto** مثبت ونشط على السيرفر الخاص بك لاستقبال وإرسال الأوامر فورياً.
* لتثبيت Mosquitto على Ubuntu/VPS:
  ```bash
  sudo apt update
  sudo apt install mosquitto mosquitto-clients
  sudo systemctl enable mosquitto
  sudo systemctl start mosquitto
  ```

### 3. تشغيل واجهة الويب الـ Frontend (React + Vite):
1. افتح نافذة أوامر جديدة في مجلد `frontend`:
   ```bash
   cd frontend
   ```
2. تثبيت الملحقات:
   ```bash
   npm install
   ```
3. تشغيل لوحة التحكم الويب محلياً للتجربة:
   ```bash
   npm run dev
   ```
*ستعمل واجهة الويب على الرابط: `http://localhost:3000`.*

## 🚀 رابعاً: الاستخدام والتهيئة الميدانية الأولى

1. عند تشغيل جهازك لأول مرة، ابحث في هاتفك عن شبكة واي فاي جديدة باسم **`Solar-Cleaner-Setup`** واتصل بها.
2. ستفتح لك تلقائياً نافذة متصفح تفاعلية (Captive Portal):
   * اضغط على **Configure WiFi**.
   * اختر شبكة واي فاي منزلك/حقلك وأدخل رقمها السري.
   * أدخل عنوان الـ VPS الخاص بك في خانة `MQTT Broker IP`.
   * أدخل معرف جهازك (مثلاً `solar_cleaner_01`) في خانة `Device ID`.
   * اضغط **Save**.
3. سيعيد الـ ESP-01 التشغيل ويتصل تلقائياً بالشبكة وبالـ MQTT Broker.
4. اذهب للموقع `http://localhost:3000` وانشئ حساب مستخدم جديد.
5. اربط جهازك الجديد بإدخال معرّفه (`solar_cleaner_01`) في لوحة التحكم.
6. **مبروك!** يمكنك الآن رؤية مؤشر المياه التفاعلي الحركي يرتفع وينخفض حياً، وجدولة أوقات التنظيف اليومية، والتحكم بالأجهزة بنقرة زر واحدة من أي مكان بالعالم! ☀️✨
